## Run Locally

Clone the project

```bash
  git clone git@github.com:IBHAGYESH/food-search-api.git
```

Go to the project directory

```bash
  cd food-search-api
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

```bash
  npm run dev
```
